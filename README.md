# Homebrew Tap for School of AI

This is the official Homebrew tap for School of AI projects.

## Installation

```bash
brew tap schoolofai/tap
```

## Formulas

### mixaroo-lite

AI-powered CLI tool that generates playlists and creates YouTube playback links.

```bash
brew install schoolofai/tap/mixaroo-lite
```

Or:

```bash
brew tap schoolofai/tap
brew install mixaroo-lite
```

## Usage

After installation:

```bash
mx-lite setup          # Configure your AI provider
mx-lite "chill lofi"   # Generate a playlist
```
